# Installation

Open a blank .txt and paste your febbox cookie inside and save it as cookie.txt
Download the Simple Http server - https://play.google.com/store/apps/details?id=com.phlox.simpleserver
Open the app and set your location to cookie.txt also remember the ip address.
Fork this repo and open showbox.js in the providers folder.
Edit the URL to the URL from step 3 (ip address) hit save.
Now you can load your own private cookie. Repeat the steps above if you want yo add multiple cookies.
